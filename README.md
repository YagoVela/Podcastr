## 💻 About

📁 Podcastr - This is a web application that allows users to listen to their favorite podcasts, see the latest releases, and get more information about the podcasts they are currently listening to.

Project developed during **5ºNLW** offered by [Rocketseat](rs).

## 🚀 Getting Started

1. Front End
2. API

### 🎲 Running the Front End

```bash
# Clone this repository
$ git clone https://github.com/YagoVela/Podcastr.git

# Go to the project folder in terminal/cmd
$ cd Podcastr

# Install the dependencies
$ yarn

# Run the application in development mode
$ yarn dev

# The server will start on port:3000 - go to http://localhost:3000
```

### 🎲 Running the API

```bash
# Clone this repository
$ git clone https://github.com/YagoVela/Podcastr.git

# Go to the project folder in terminal/cmd
$ cd Podcastr

# Install the dependencies
$ yarn

# Run the application in development mode
$ yarn server

# The server will start on port:3333 - go to http://localhost:3333
```
